## Viewing Diagrams
To view the .mmd diagram files:
1. Install Mermaid CLI: `npm install -g @mermaid-js/mermaid-cli`
2. Convert to PNG: `mmdc -i diagram.mmd -o diagram.png`
3. Or use online Mermaid editor: https://mermaid.live/
