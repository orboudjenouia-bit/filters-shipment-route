# User Manual - Wesselli Logistics Platform

## Getting Started

### Registration
1. Visit the app and click "Get Started"
2. Choose account type: Individual or Business
3. Fill registration form:
   - Email and password
   - Phone number
   - Type-specific details (name/NIN for individuals, company details for businesses)
4. Check email for verification code
5. Enter 6-digit code to verify account

### Login
1. Enter email and password
2. Click "Login"
3. Access dashboard

## Dashboard
View overview of:
- Active shipments
- Available routes
- Recent activity
- Account statistics

## Managing Shipments

### Creating a Shipment
1. Go to "Shipments" tab
2. Click "Create Shipment"
3. Fill details:
   - Title and category
   - Origin and destination
   - Weight and volume
   - Date and time
   - Priority level
   - Special instructions
4. Upload photo (optional)
5. Submit

### Viewing Shipments
- Filter by status: All, Active, Pending, Completed
- Click shipment for details
- Edit or delete your shipments

## Managing Routes (For Carriers)

### Creating a Route
1. Go to "Routes" tab
2. Click "Create Route"
3. Select vehicle
4. Set origin, destination, and dates
5. Add route details and photo
6. Publish route

### Viewing Routes
- See all your routes
- View assigned shipments
- Update route status

- View all users

## Troubleshooting

### Can't Login?
- Check email verification
- Reset password if forgotten
- Ensure correct email/password

### Shipment Not Appearing?
- Check internet connection
- Refresh page
- Contact support if persists

### Route Assignment Issues?
- Verify vehicle availability
- Check date conflicts
- Ensure profile is complete
